# 无障碍服务快速使用指南

## 📱 用户使用流程

### 第一步：点击功能卡片
在主屏幕上点击任意需要无障碍权限的功能卡片，例如：
- 自动任务
- 自动连点器
- 自动滚动
- 自动刷新

### 第二步：查看提示
如果无障碍服务未开启，会看到一个对话框：

```
标题：需要开启无障碍服务
内容：自动任务需要无障碍服务权限才能正常工作。
      请在设置中为本应用开启无障碍服务。
按钮：[取消] [去设置]
```

### 第三步：前往设置
1. 点击 **"去设置"** 按钮
2. 系统会自动跳转到 **无障碍服务设置** 页面
3. 会弹出一个操作提示：

```
标题：操作提示
内容：请在无障碍设置中找到"白虎"应用，并开启服务开关
按钮：[知道了]
```

### 第四步：开启服务
在系统设置中：
1. 找到 **"白虎"** 或 **"baihu"** 应用
2. 点击进入
3. 开启 **无障碍服务开关**
4. 阅读并同意权限说明
5. 点击 **"确定"** 或 **"允许"**

### 第五步：返回应用
1. 返回到白虎应用
2. 会自动检测到服务已开启
3. 显示成功提示：

```
标题：无障碍服务已启用
内容：现在可以使用自动任务功能了！
按钮：[知道了]
```

### 第六步：开始使用
现在所有需要无障碍服务的功能卡片都已启用，可以正常使用了！

## ❓ 常见问题

### Q1: 为什么需要无障碍服务？
A: 无障碍服务允许应用程序自动执行点击、滑动等操作，这是实现自动化任务的必要权限。

### Q2: 无障碍服务安全吗？
A: 白虎应用仅使用无障碍服务来执行您录制的自动化任务，不会收集或上传任何敏感信息。

### Q3: 找不到白虎应用怎么办？
A: 在无障碍设置页面：
1. 查找 "白虎" 或 "baihu"
2. 如果列表很长，尝试使用搜索功能
3. 确保应用已正确安装

### Q4: 开启后为什么还是提示未启用？
A: 可能的原因：
1. 系统需要几秒钟才能生效，请稍等片刻
2. 尝试完全关闭应用后重新打开
3. 在系统设置中确认服务确实已开启

### Q5: iOS 设备需要开启吗？
A: 不需要！无障碍服务检测仅在 Android 平台生效，iOS 设备会自动跳过检查。

### Q6: 会影响其他应用吗？
A: 不会。无障碍服务仅在白虎应用运行时工作，不会影响其他应用的正常使用。

### Q7: 如何关闭无障碍服务？
A: 如果您不再需要使用自动化功能：
1. 进入系统设置 → 无障碍
2. 找到白虎应用
3. 关闭服务开关

## 💡 温馨提示

1. **首次使用**：第一次使用自动化功能时需要开启，之后就不需要重复操作
2. **系统更新**：某些系统更新可能会重置权限，如遇到问题请重新开启
3. **省电模式**：部分设备的省电模式可能会限制无障碍服务，使用时建议关闭省电模式
4. **权限管理**：不同品牌手机的设置界面可能略有差异，但基本流程相同
5. **安全提示**：只从官方渠道下载白虎应用，避免使用来路不明的版本
